We need to do some string substitutions:
– There’s an “m” for million in the first row of the Population. We need to make it into something that can be parsed as a number.
(Since the column is still of type character and not numeric, we can just replace it with scientific notation characters.)
– We want to do a similar thing for “bn” for billion in the GDP columns.
– We also want to take out the out € symbol
So, we’ll make a custom gsub function returns a tibble.

multi_sub <- function(input, output, dat) {
  result <- dat
  for (i in 1 : length(input)) {
    result <- as_tibble(as.data.frame
                        (sapply(dat, gsub, 
                                pattern=input[i], replacement=output[i], 
                                   result)))
  }
  result
}

Make our replacements

em_tab <- multi_sub(input = " m", output = "e+06", dat = em_tab)
em_tab <- multi_sub(input = "bn", output = "", dat = em_tab)
em_tab <- multi_sub(input = "€", output = "", dat = em_tab)

Adventures in Text Munging and Wordclouding

Scrape all text (excluding citations) from the same Wikipedia page. Note that we’re using the selector p in html_nodes(). If we use #bodyContent instead, the citations at the bottom of the page get included, which we don’t want.

  wiki_page %>% 
  html_nodes("p") %>%  
  html_text
